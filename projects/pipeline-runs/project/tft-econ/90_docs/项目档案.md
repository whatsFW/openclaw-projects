## 核心功能

1. **经济建模** — 纯经济变量模拟（基础金币 + 利息 + 连胜/连败奖励）
2. **策略对比** — 蒙特卡洛模拟多种策略的经济收益
3. **交互可视化** — 参数调整即时反馈（< 200ms），多策略曲线叠加对比
4. **技术分析文档** — 结构严谨的分析报告
