## 核心功能

1. **登录/注册** — 模拟校验（任意非空通过），注册后跳登录并自动填充
2. **店铺列表** — 8-12 家 Mock 店铺卡片展示
3. **店铺详情** — 菜品列表 + "+" 加入购物车
4. **购物车** — 增删改数量 + 价格计算 + localStorage 持久化 + 店铺隔离
5. **确认订单** — 汇总 + 默认地址 + 提交 → Toast → 跳转
6. **拟物化 UI** — 按钮/输入框/卡片/导航栏统一 Design Token
